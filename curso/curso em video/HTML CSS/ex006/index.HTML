Lista

Ordenadas

São listar que a ordem dos item importam.
Para criar uma ordered lists usamos a tag <ol> para delimitar a lista e a tag <li> para identificar cada item da lista.

  1. água gelada
  2. suco solúvel
  3. misturar bem
  4. esta pronto

A tag <ol> possui uma parâmetro type, onde configuramos o tipo de marcador da lista atual. As opçoes de valores para essa parâmetro são:

→ 1-Valor padrão. Criar listas numeradas. Ex:1,2,3,4,
→ A-Criar listas alfabéticas em maiúsculas.Ex A,B,C,D,...
→ a-Criar listas alfabéticas minúsculas.Ex a,b,c,d...
→ I-Crar listas com algarismos romanos m maiúslas. Ex I,II,III,IV,....
→ i- Criar listas com algarismos romanos em minúsculas. Ex i,ii,iii,...
Você também pode indicar o inicio da contagem usando parâmetros star.
Por exemplo, a tag <ol type="I" start = "5"> vai gerar itens numerados como V,VI ,VIII, XI,....

Não ordenadas

Ela é apenas uma ótima maneira para organizar os itens que não apresentam uma classificação necessariamente.
Para criar uma unordered list, vamos usar a tag <ul> para delimitar a lista e a tag <li> para criar cada uma dos seus itens internos.

  • Iêdson
  • Ieda
  • Ingrid

O marcador padrão é a bolinha preta totalmente preenchida (circle), mas exite a opção de configurar a propriedade type da tag <ul> com os seguintes valores:
→ disc-Padrão. Uma bola preta tolamente pintada
→ circle-Uma bola com uma borda preta e sem preenchimento.
→ square- Um pequeno quadrado preto totalmente pintado.

Misturando as coisas

Podemos também criar listas mistas, configurando listas dentro de outras listas.

  1. item 1
  2. item 2
    1. item 2.1
    2. item 2.2
    3. item 2.3
  3. item 3
  4. item 4
  5. item 5

Lisata de Definição

É como se fosse um dicionaáio, temos e as suas descrições . É uma lista sem demarcadores, mas bem útil em alguns casos. Toda lista de definições está dentro de uma tag <dl>(definition list) cada termo é um <dt>(definition term) e cada descrição é um <dd> (definition description).
